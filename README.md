# p5-sketches

> my development environment for [p5.js](https://p5js.org/) sketches

## Installation

```sh
yarn install
```

## Create new sketch

```sh
yarn new my-awesome-sketch
```

## Development

```sh
yarn dev sketches/my-awesome-sketch
```

## Build

```sh
yarn build
```

## Lint

```sh
yarn lint
```

## Format

```sh
yarn format
```
